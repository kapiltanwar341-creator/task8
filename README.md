# Task 8 — Create a Simple Blog Layout Using Bootstrap 5

## Objective

Use **Bootstrap 5 CDN** to build a responsive blog page with a **navbar**, **card-based posts**, and a **footer**.

## Tools

* VS Code (or any editor)
* Modern browser (Chrome, Firefox)
* Bootstrap 5 CDN

---

## Example project structure

```
bootstrap-blog/
├── README.md
├── index.html
├── assets/
│   ├── hero.jpg
│   └── post-1.jpg
└── custom.css   (optional)
```
